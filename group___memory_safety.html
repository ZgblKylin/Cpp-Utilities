Detailed Description

Helper classes, typedefs and functions for memroy safety.

Provide helperful functionalities for memory safety with:

  • Memory::RWSpinLock : A extremely high-performance read-write-spinlock imported from folly library.
  • Memory::SafeSharedPtr : A wrapper to std::shared_ptr to provide thread-safety while operating the underlying pointer.
  • Memory::SafeWeakPtr : A wrapper to std::weak_ptr to cooperate with Memory::SafeSharedPtr.
