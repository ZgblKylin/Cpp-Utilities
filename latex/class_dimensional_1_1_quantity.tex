\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T, typename U, typename Ratio = std\+::ratio$<$1$>$$>$\newline
class Dimensional\+::\+Quantity$<$ T, U, Ratio $>$}
The \mbox{\hyperlink{class_dimensional_1_1_quantity}{Quantity}} struct is used to describe arithmetic values with units. 


\begin{DoxyTemplParams}{Template Parameters}
{\em T} & Arithmetic type for value. \\
\hline
{\em U} & \mbox{\hyperlink{struct_dimensional_1_1_unit}{Unit}} type for this physical quantity. \\
\hline
{\em Ratio} & Conversion ratio for nonstandard units such as feet or yard.\\
\hline
\end{DoxyTemplParams}
This template guarantee strong-\/typed safe calculation of physical values.~\newline
 Variables with different unit cannot add, subtract and compare with each other. Multiply, divide, power, root calculation will generate value with unit.~\newline
 Value is allowd to be described with different Ratio. Calculation with different ratios is safe, and result has same Ratio of first operand.~\newline
