\begin{DoxyNote}{Note}
Copy constructor and copy assignment are deleted to prevent multiply locks, use {\ttfamily std\+::move} with move constructor and move assignment to transport it\textquotesingle{}s ownership, or simply use it like type T$\ast$ or T\&.~\newline
 Because operator. cannot be overloaded, {\ttfamily p\+Points\mbox{[}0\mbox{]}.x} cannot compile, use {\ttfamily Point(p\+Points\mbox{[}0\mbox{]}).x} instead. Sorry for that. 
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
Behavior is undefined if {\ttfamily T} is not array type. 
\end{DoxyWarning}
