Try to get reader permission on the lock. This can fail if we find out someone is a writer or upgrader. 

Setting the UPGRADED bit would allow a writer-\/to-\/be to indicate its intention to write and block any new readers while waiting for existing readers to finish and release their read locks. This helps avoid starving writers (promoted from upgraders). \mbox{\Hypertarget{class_memory_1_1_r_w_spin_lock_ab47cfb0f09949a579afcfc6018b04297}\label{class_memory_1_1_r_w_spin_lock_ab47cfb0f09949a579afcfc6018b04297}}

\begin{DoxyNote}{Note}
